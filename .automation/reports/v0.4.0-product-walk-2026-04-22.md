## P0 阻断级问题

### P0-A：13-Agent 推荐"完成"语义错乱
- **现象**：20 条历史推荐中唯一一条标记为「完成」的，实为失败 694 秒；点开后 Agent 反向追问用户给出意见。
- **复现**：进入「13-Agent 推荐」→ 历史列表 → 点击唯一「完成」状态条目。
- **建议修法**：分清「完成 / 失败 / 用户中止」三态；失败状态绝不可标记成完成；详情页不应让 Agent 反问用户。
- **影响**：用户对推荐模块的信任会被永久性破坏。

### P0-B：情绪轮动核心榜单全线"数据不可用"
- **现象**：综合强度 = 0.0，扩散 / 持续指数空，所有维度榜单空白。
- **复现**：进入「情绪轮动」首页。
- **建议修法**：先排查上游 `BoardSnapshot` / 行业涨幅快照是否真的为空，再决定是后端数据缺失还是前端渲染兜底缺失；至少要给空态友好文案与最近一次有效快照时间。
- **影响**：模块视觉上等同于挂掉。

## P1 高优先级问题

### P1-C：股票详情基本面 3 字段空
- **现象**：流通市值 / 股东户数 / 所属板块 三字段在股票详情页空缺。
- **建议修法**：核对腾讯 / 东财对应 endpoint 字段名与单位；缺失时显示 `--` 而不是空白。

### P1-D：主力净流入跨页矛盾（疑单位 bug）
- **现象**：股票信息页显示 -69.22 亿，情绪轮动页显示 -0.0 亿，疑为单位换算 bug（万 ↔ 亿 ↔ 元）。
- **建议修法**：抽出统一的 `formatMoneyYi(value, sourceUnit)` 工具，跨页强制走同一格式化路径。

### P1-E：Worker 运行 17 分钟但日志面板 0 条
- **现象**：Worker 持续运行 17 分钟，UI 日志面板始终 0 条。
- **建议修法**：核对 Serilog sink → SignalR/SSE → 前端订阅链路；确认 sink 没有被环境变量关掉。

### P1-F：失败按钮"重新连接"语义不友好
- **现象**：推荐失败后按钮文案写「重新连接」，给用户的语义是「网络断了」，但实际是「重新生成推荐」。
- **建议修法**：文案改为「重新推荐」或「再试一次」。
