# GOAL-017 Plan / 量化双引擎与 Agent/图表协同规划

## Plan (EN)
- Objective: define a production-friendly quant capability layer that connects charting, agent reasoning, and trading-plan workflows without turning the main runtime into a heavyweight quant platform.
- Decision: use `Skender primary + Lean shadow`.
  - `Skender` (or equivalent lightweight .NET indicator library) becomes the online primary engine for feature extraction and strategy signals.
  - `Lean` becomes the shadow/replay/calibration engine for parity checks, replay, backtesting, and acceptance baselines.
- Core architecture:
  - normalized market-data layer first
  - unified feature/signal contracts second
  - primary-only default consumption in chart/agent/plan flows
  - shadow results reserved for developer mode, replay, calibration, and research
- Planned slices:
  - R1 normalized input layer + quant contracts
  - R2 Skender primary runtime integration
  - R3 Lean shadow replay/calibration integration
  - R4 product integration across chart, MCP, agent, and trading-plan flows

## 计划（ZH）
- 目标：定义一层适合当前产品形态的量化能力，把图表、Agent 推理和交易计划真正打通，同时避免把主运行时直接拖成重型量化平台。
- 决策：采用 `Skender primary + Lean shadow`。
  - `Skender` 或同类轻量 .NET 指标库作为在线主引擎，负责特征提取和策略信号。
  - `Lean` 作为影子/回放/校准引擎，负责一致性比对、回测、replay 与验收基线。
- 核心架构：
  - 先统一行情输入归一化层
  - 再统一 feature/signal contract
  - 图表/Agent/交易计划默认只消费 primary 结果
  - shadow 结果主要进入开发者模式、回放、校准与研究链路
- 规划切片：
  - R1 归一化输入层 + 量化 contract
  - R2 Skender 主运行时整合
  - R3 Lean 影子 replay/calibration 整合
  - R4 图表、MCP、Agent、交易计划产品层整合

## Risks / Notes (EN)
- The most important boundary is not library choice alone, but the single-source-of-truth rule: shadow output must not silently become a second online decision source.
- Same-name indicators across Skender and Lean may disagree because of warmup or data-handling differences; this is expected and should be tracked explicitly through comparison contracts.

## 风险 / 备注（ZH）
- 最关键的边界不是单纯选库，而是“单一主口径”：shadow 结果不能悄悄变成第二套线上决策源。
- Skender 与 Lean 的同名指标可能因 warmup 或数据处理差异而不完全一致；这属于预期行为，应通过 comparison contract 显式记录，而不是假设天然相同。
