# GOAL-012-R1 Chart Terminal Extensibility Upgrade - Development Report (2026-03-14)

## EN
### Summary
Completed the Dev2 isolated frontend refactor for the stock terminal chart area without touching Dev1 Step 4.2 trading-plan contracts. The chart area now uses a single main viewport with unified tabs `分时图 / 日K图 / 月K图 / 年K图`, and the internal rendering logic was moved behind a reusable chart adapter layer under `frontend/src/modules/stocks/charting/**`.

### Selection Result
- Evaluated the upgrade target as a TradingView-style interaction model rather than library swap for its own sake.
- Chose to keep `lightweight-charts` for this iteration because it is already integrated, Vue 3-safe in this repo, and preserves data compatibility with current `minuteLines` / `kLines` payloads.
- Added an internal adapter layer instead of introducing a new dependency so future overlays like TD Sequential, KDJ crossover, or custom markers can be attached without destabilizing the current stock detail flow.

### Code Changes
- Rebuilt `frontend/src/modules/stocks/StockCharts.vue` into a single-viewport tabbed terminal.
- Added `frontend/src/modules/stocks/charting/chartViews.js` to centralize chart-tab definitions and supported K-line intervals.
- Added `frontend/src/modules/stocks/charting/useStockChartAdapter.js` to isolate `lightweight-charts` creation, rendering, resize, and overlay plumbing.
- Updated `frontend/src/modules/stocks/StockCharts.spec.js` with unified-tab behavior coverage.
- Synced `README.md`, `.automation/tasks.json`, and `.automation/state.json` to reflect GOAL-012-R1 progress.

## ZH
### 摘要
已完成 Dev2 独立前端改造，在不触碰 Dev1 Step 4.2 交易计划 contract 的前提下重构股票终端图表区。图表区现已改为单主图视口 + 统一 Tab：`分时图 / 日K图 / 月K图 / 年K图`，并把内部渲染逻辑抽离到 `frontend/src/modules/stocks/charting/**` 适配层中。

### 选型结论
- 本轮目标是获得 TradingView 风格的可扩展交互，而不是为了换库而换库。
- 继续保留 `lightweight-charts`，原因是它已在本仓库稳定接入、对 Vue 3 兼容成熟，且能直接复用现有 `minuteLines` / `kLines` 数据结构。
- 通过新增内部 adapter 层而不是引入新依赖，为后续神奇九转、KDJ 金叉、自定义标记等叠加图层预留接入点，同时避免冲击现有股票详情与交易计划流。

### 代码变更
- 将 `frontend/src/modules/stocks/StockCharts.vue` 改造成单主图 Tab 终端。
- 新增 `frontend/src/modules/stocks/charting/chartViews.js`，集中管理图表 Tab 定义与支持的 K 线周期。
- 新增 `frontend/src/modules/stocks/charting/useStockChartAdapter.js`，隔离 `lightweight-charts` 的创建、渲染、resize 与叠加图层管线。
- 更新 `frontend/src/modules/stocks/StockCharts.spec.js`，补充统一 Tab 行为覆盖。
- 同步更新 `README.md`、`.automation/tasks.json`、`.automation/state.json` 反映 GOAL-012-R1 当前进度。
