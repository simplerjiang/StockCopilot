## EN
### Planning
- Goal: remove the low-hit-rate sector-context path that reused Sina 7x24 macro rolls and replace it with a sector-targeted source plus a richer global macro base.
- Source validation completed before coding:
  - CNBC Finance RSS returned standard XML successfully.
  - WSJ US Business RSS returned standard XML successfully.
  - NYT Business RSS timed out in the current environment, so the implementation keeps multi-source aggregation plus deterministic fallback.
  - The provided Eastmoney sector-search URL redirected to the Eastmoney portal in this environment, so the parser was implemented defensively to accept multiple JSON shapes and safely degrade on HTML/non-JSON responses.

  - Added `RssMarketNewsParser` and aggregated CNBC/WSJ/NYT feeds for `level=market` ingestion.
  - Added a deterministic fallback to Sina roll only when all global RSS feeds fail.
- Tests added/updated:
  - `LocalFactIngestionServiceTests`
  - `EastmoneySectorSearchParserTests`
  - `RssMarketNewsParserTests`
- Regression coverage:
