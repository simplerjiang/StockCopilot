# Stock Copilot v0.0.2

一个面向 A 股场景的本地优先桌面助手。

它现在不只是“查行情 + 跑一次 AI 分析”的 Demo，而是已经把股票终端、多 Agent 分析、市场情绪、本地资讯库、交易计划草稿和桌面打包链路收进了一个统一产品里。目标很直接：少切窗口，少看噪音，更快进入判断和执行前准备。

完整开发说明、任务拆解、技术细节和自动化记录，请看 README.llm.md。

## v0.0.2 最新功能

- 股票终端已经形成完整工作台：支持分时、日 K、月 K、年 K，以及策略指标和图表叠加
- 多 Agent 分析已经从“单句结论”升级为“观点 + 条件 + 风险 + 证据”的结构化输出
- 情绪轮动页已经支持市场阶段、主线板块、比较窗口、实时总览和板块榜联动
- 本地资讯库已经支持个股、板块、市场多层级资讯聚合和页面展示
- 交易计划已经具备草稿、总览、提醒、市场上下文联动等基础闭环
- Stock Copilot 的 MCP 运行时底座已经落地，包含 K 线、分时、策略、新闻、搜索五类受控工具接口
- Stock Copilot 会话草案能力已经落地，能够返回 planner / governor 时间线、tool call 草案、follow-up action 和待执行 final answer contract
- Windows 桌面打包链路已经稳定，安装版会自动拉起后端，默认使用本地 SQLite 运行，不要求用户再手工装数据库
- 发布版已经接上 GitHub Releases 更新检测，支持发现新版本并触发安装器升级

## 核心能力

- 看盘：一页内完成查股、看图、看分时、切周期、看关键指标
- 分析：多 Agent 输出核心观点、触发条件、失效条件、风险上限与证据来源
- 看市场：用情绪轮动页集中查看市场阶段、主线板块、涨跌分布和资金温度
- 看资讯：把零散新闻沉淀到本地资讯库，并按个股、板块、市场分层呈现
- 做计划：从分析结果出发生成交易计划草稿，并保留后续复核和提醒空间
- 跑桌面：安装后即可使用，不需要自己分别起前后端和数据库

## 怎么安装

推荐直接从 GitHub Releases 下载 v0.0.2 对应安装包：

- SimplerJiangAiAgent-Setup-v0.0.2.exe

发布页：<https://github.com/simplerjiang/AiAgent/releases>

如果你不想安装，也可以下载便携包：

- SimplerJiangAiAgent-portable-v0.0.2.zip

## 第一次打开怎么用

1. 安装后启动程序。
2. 如果还没有配置可用 LLM Key，程序会自动跳到 LLM 设置。
3. 用管理员账号登录。
4. 填写并保存可用的接口地址、模型和 API Key。
5. 回到股票终端，开始查股票、看图和运行多 Agent 分析。

## 默认 LLM 接口说明

当前 README 中展示或示例使用的默认接口地址是：

- https://one-api.bltcy.top/

这个地址来自第三方网站，不是本项目官方自建服务。

你可以继续使用该接口，也可以在 LLM 设置里改成你自己的 One API、中转服务或其它兼容 OpenAI 接口格式的提供方。项目本身不强绑定任何单一地址，真正决定是否可用的是你填写的接口地址、模型名称和 API Key。

当前默认管理员账号：

- 用户名：admin
- 密码：admin123

## 自动更新

发布版桌面程序已经接上 GitHub Releases 更新检测：

- 启动后会检查是否存在新版本
- 检测到新版本时会弹窗提醒
- 确认后会自动下载安装器并执行升级

本地数据、日志和 LLM 配置仍然保存在当前 Windows 用户目录，不会因为升级被覆盖。

## 数据放在哪

打包版默认会把运行数据放到：

- %LOCALAPPDATA%\SimplerJiangAiAgent

这样做的好处：

- 数据和程序本体分开，升级更稳
- 换版本时不容易覆盖用户的本地配置和使用痕迹

## 本地打包后启动

如果你在源码仓库里想直接验证“GitHub 用户最终下载到的那个桌面 EXE”是否能用，可以在项目根目录运行：

```powershell
.\start-all.bat
```

这个脚本会：

- 停掉当前仓库残留的桌面程序和后端进程
- 运行 scripts\publish-windows-package.ps1 打出最新 artifacts\windows-package
- 直接启动 artifacts\windows-package\SimplerJiangAiAgent.Desktop.exe

它验证的是打包后的桌面程序，而不是只供开发调试的浏览器页面。
